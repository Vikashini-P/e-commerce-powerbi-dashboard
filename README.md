# E-commerce Sales Analytics Dashboard (Power BI)

## Overview
This project presents an interactive E-commerce Sales Analytics Dashboard built using Power BI to analyze sales performance, category trends, order status, and key KPIs.

## Dashboard Pages
- Sales Overview
- Category Analysis
- Order Status Analysis
- KPI Overview

## Key Features
- Monthly sales and order trends
- Category-wise sales and contribution analysis
- Order status distribution and sales impact
- KPI metrics: Total Sales, Total Orders, Average Order Value
- Date and Category slicers synced across all pages

## Tools & Skills
- Power BI Desktop
- Data Visualization
- Dashboard Design
- Business Analytics
- Interactive Reporting

## Files
- `.pbix` file: Complete Power BI dashboard
- PDF report: Detailed project documentation

## Outcome
The dashboard enables data-driven decision-making by providing clear and actionable insights into e-commerce sales performance.
